# llm-competitive-intelligence-dashboard
Automated competitive intelligence tool that uses multi-step LLM workflows to analyze competitor websites, extract structured data, and generate comparison reports.

## What it does

- Scrapes competitor websites
- Uses LLM to select relevant pages (products, pricing, features)
- Extracts structured competitive data
- Generates comparison reports

## Installation
bash
pip install -r tiktoken openai python-dotenv beautifulsoup4 requests ipython
Set up your '.env' file with your OpenAI API key.
